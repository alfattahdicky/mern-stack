# Mern Stack

## Tech Stack
- MySQL
- Node JS
- Express
- React JS

## Feature
- Create Register Page Home
- Read, Update & Delete Page Admin


## How to Run
1. Create Database "multimatics_test" on MySQL

2. Run backend folder (localhost:8085)
```
  npm run dev
```
2. Run folder frontend (localhost:5173)
```
  npm run dev
```
